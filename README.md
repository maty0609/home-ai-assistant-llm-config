# Home AI Assistant - LLM Provider Configuration

This repository contains the implementation of configurable LLM providers for the Home AI Assistant backend, supporting Azure OpenAI, OpenAI, and Local OpenAI models.

## Features

- Dynamic LLM provider selection based on environment variables
- Support for multiple providers (Azure OpenAI, OpenAI, Local OpenAI)
- Backward compatibility with existing configurations
- Comprehensive documentation for configuration

## Configuration

See [LLM_PROVIDER_CONFIGURATION.md](LLM_PROVIDER_CONFIGURATION.md) for detailed information on configuring different LLM providers.
